* Arduino Leonardo
* Servo
* Motor
* Ultrazvukové čidlo

# Zapojeni pinu
 + D2-9 klavesnice
 + D10,11 motor 1
 + D12,13 motor 2
 + A4 rele pro magnet
 + A0,1 ultrazvukove cidlo
 + A2,3 ultrazvukove cidlo 2
